Class MovieCardComponent

This is the component for the home page; it allows the user to view the list of movies, add movies to their list of favorites, and view each movie's details.

MovieCardComponent retrieves the full list of movies from the API, allows the user to add or remove a movie from their list of favorites, and opens dialog windows with different details of each movie.
